# vue3.0组件
***************************************
采用`vue3.0`的最新语法，写一些小组件来练练手。目前用的css，后期考虑用scss的模板

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
